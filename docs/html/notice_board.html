notice_board module¶

This program will show all posted notices in our Office Management System. There are four functions to group them for the user help.
Author: Ashiqur Rahman
ID: 1712389642

Functions:

notice_board.allNotices()[source]¶

shows all notices form the database and renders to Office Management System’s Notice Board page.

notice_board.groupByDate()[source]¶

groups all the notices by present date and shows notices posted today.

notice_board.groupByDept(dpt)[source]¶

groups notices by department and shows selected notices.

  • Attribute: dpt
  • Type: string
  • Description: desired department name to see department associated notices.

notice_board.groupByPriority(pt)[source]¶

groups notices by their priority level and shows them.

  • Attribute: pt
  • Type: string
  • Description: desired priority level to see associated notices.
